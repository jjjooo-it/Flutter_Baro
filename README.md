## 📢프로젝트 설명
'Tmax AI Bigdata Academy 4기'실무중심종합설계 프로젝트로 진행한 프로젝트입니다.
'BARO'는 

- 핵심 기능
  - 크롤링 한 대피소 정보를 지도에 마킹하여 한 눈에 보여주기
  - 현재 위치 기반 가장 가까운 대피소 안내하기
  - 지진 발생 시 알림 및 발생 위치 마킹하기 
  - 지진 관련 실시간 뉴스 AI 요약해서 제공하기
